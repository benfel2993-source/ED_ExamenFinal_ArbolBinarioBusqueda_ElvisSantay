# Árbol Binario de Búsqueda (BST) en Java

## Descripción
Este proyecto implementa desde cero un Árbol Binario de Búsqueda (BST) en Java, sin utilizar librerías externas de estructuras de datos.

El objetivo es comprender el funcionamiento interno de los árboles binarios mediante la implementación manual de sus operaciones principales.

---

## Funcionalidades

El sistema permite:

- Insertar nodos (Insert)
- Buscar valores (Search)
- Eliminar nodos (Delete)
- Recorridos del árbol:
  - In-Order
  - Pre-Order
  - Post-Order

---

## Estructura del proyecto

- `Nodo.java` → define la estructura de cada nodo del árbol
- `BST.java` → contiene toda la lógica del árbol (insertar, buscar, eliminar, recorridos)
- `Main.java` → archivo de pruebas y ejecución del programa

---

## Cómo ejecutar

```bash
javac src/*.java
java -cp src Main
```
